<p>HappyColours is a personal experiment to see if concentrating at looking at specific colours makes the user feel happier.</p>
<p>In the app there are four gray squares, which change colour occasionally, the player gets ”points” by pressing the square that’s not currently gray.</p>
<p></p>The colours used in HappyColours are based on the findings that were made in Feeling Blue or Seeing Red? Similar Patterns of Emotion Associations With Colour Patches and Colour Terms study (https://www.ncbi.nlm.nih.gov/pmc/articles/PMC7027086/). 
The study was done by Domicele Jonauskaite, C. Alejandro Parraga, Michael Quiblier, and Christine Mohr. I converted the CIE colours used in the study to 0–1 RGB, 
so it’s likely they aren’t an exact match, but close enough.</p>

<p>One has to admit that a fullscale literature review would be in place to get more comprehensive results regarding the proper colour choises, 
but considering that this is a prototype, the level of background study is adequete–at least in my mind.</p>

<p>Please note that COLOUR FEED scene does exists in the project files, but it's not currently in use. In ColourFeed the user scrolls through randomly generated colours and each colour has a question or thought attached to them. ColourFeed is not currently accessible in the app because, but is "playable" in Unity.</p>
<b>Colour Feed</b>
<p>
Colour Feed is an experiment to see if browsing through different random colours and reading text snippets attached to them helps with mindfullnes. 
Currently there’s only one static text attached to each colour, but randomized text are in the works.</p>
